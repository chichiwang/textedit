    Objectives of this exercise

      1. Develop from a template:
        - Build a template with elegant directory structure
          (layout-specific files in an assets directory, etc.)
        - HTML5 Shiv
        - Normalize CSS3
        - Load jQuery from the Google CDN
          (also link to a local copy)

      2. Become familiar with front-end development tools:
        - CSS3 PIE
        - Colorzilla CSS3 Gradient Editor
        - jQuery UI
          (slider widget)

      3. Practice front-end techniques:
        - Clean, structured CSS rules
        - Form styling
        - CSS font loading
          (@font-face and font kits)
        - Object approach to CSS classes
          (tabs, buttons, page elements, etc.)
        - CSS3 gradient syntax
        - jQuery Plugin Authoring
          (write plugins for menus and text-edit functionality)
        - Simple regular expressions
        - Dropdown menus built from scratch
        - Clean, readable, modular javascript

      4. Adaptive Design:
        - Responsive elements for all screen devices
        - Adaptively designed layout
          (aim to provide the best experience for each device)
        - Touch device feature detection in javascript

      5. Sweet ass design:
        - Make her look slick
